Автопрайсер для маркетплейса

Сервис сам считает цены от себестоимости, маржи и коридора — и обновляет их на ~5500 товарах.

живое демо — двигайте параметры, цены пересчитываются сразу // настоящая математика на моковых товарах
Параметры ценообразования
Коридор цен — защита от крайностей
{{ avgMargin }} % средняя фактическая маржа
{{ clampedCount }} товаров упёрлись в коридор
Товар Себест. Комиссия Цена Маржа Статус
{{ r.name }} {{ r.cost }} ₽ {{ r.comm }} % {{ r.price }} ₽ {{ r.marginPct }} % {{ r.status }}

В демо — 8 товаров. В рабочей версии тот же расчёт идёт по ~5500 позициям: себестоимость и комиссии подтягиваются из Ozon API, цены обновляются по расписанию.

01 · задача

Держать актуальные цены на тысячах товаров вручную невозможно: меняются себестоимость, комиссии и логистика — маржа уплывает незаметно.

02 · как решал

Сервис на Python: тянет товары и комиссии из Ozon API, считает цену от себестоимости, желаемой маржи и коридора, обновляет по расписанию и пишет отчёт об изменениях.

03 · результат

~5500 товаров переоцениваются автоматически. Ручная работа с ценами свелась к настройке параметров.
