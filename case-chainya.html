Задача

У локальной чайной несколько разных аудиторий: гости зала, покупатели чая домой и бизнес-клиенты. Нужно было объяснить непривычный формат церемонии, показать ассортимент и привести каждого к своему действию, сохранив камерный характер бренда.

Решение

Собрали одну быструю страницу с отдельными маршрутами: забронировать стол, выбрать чай, запросить поставку. Каталог фильтруется по типам, заявки уходят в Telegram, а язык и тема переключаются без перезагрузки.

Почему без фреймворка

Для этого проекта важнее скорость загрузки и простое обслуживание, чем тяжёлая CMS. Весь интерфейс работает на HTML, CSS и JavaScript без внешних зависимостей; изображения и шрифты оптимизированы и кэшируются отдельно.

Что осталось у бизнеса

Готовый сайт на собственном домене, адаптивная мобильная версия, SEO-разметка локального бизнеса, Telegram-бот для заявок и исходники без привязки к конструктору или ежемесячной лицензии.
